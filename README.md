# 2D 탱크 배틀 게임 🎮

클래식 아케이드 스타일의 2D 탱크 전투 게임입니다. 마우스로 조준하고, 폭발 효과가 있는 총알로 적 탱크를 제거하세요!

## 주요 기능

- **마우스 조준 시스템**: 마우스 커서 방향으로 탱크가 회전하고 클릭하여 발사
- **폭발 메커니즘**: 총알이 적이나 벽에 맞으면 폭발하여 범위 데미지 발생
- **자폭 위험**: 폭발 범위에 플레이어가 있으면 자신도 피해를 입음
- **AI 적 탱크**: 자동으로 이동하고 플레이어를 추적하여 공격
- **파괴 가능한 벽**: 벽돌 벽은 파괴 가능, 강철 벽은 파괴 불가
- **레벨 시스템**: 모든 적 처치 시 다음 레벨 진행, 난이도 증가
- **파워업 아이템**: 체력 회복 및 공격력 증가 아이템
- **파티클 효과**: 폭발, 충격, 연기 효과
- **모바일 지원**: 터치 조이스틱과 발사 버튼

## 게임 플레이

### 목표
각 레벨의 모든 적 탱크를 처치하여 다음 레벨로 진행하세요. 체력이 0이 되면 게임 오버입니다.

### 조작 방법

#### 데스크톱
- **W, A, S, D**: 탱크 이동 (앞, 왼쪽, 뒤, 오른쪽)
- **마우스 이동**: 탱크 포탑 조준
- **마우스 클릭**: 발사
- **P**: 일시정지
- **R**: 게임 재시작

#### 모바일
- **왼쪽 조이스틱**: 탱크 이동
- **화면 터치**: 해당 방향으로 조준 및 발사
- **발사 버튼**: 현재 조준 방향으로 발사

## 게임 요소

### 탱크 종류
- **플레이어 탱크** (파란색): 3 HP, 마우스 조준, 폭발 총알
- **적 탱크** (빨간색): 2 HP, AI 제어, 자동 추적 및 발사

### 장애물
- **벽돌 벽** (갈색): 파괴 가능, 총알 3발로 파괴
- **강철 벽** (회색): 파괴 불가능, 영구적 장애물

### 파워업
- **체력 회복** (초록색 +): HP 1 회복
- **공격력 증가** (빨간색 별): 데미지 2배 (5초간)

### 폭발 시스템
- **폭발 범위**: 충돌 지점 기준 반경 60픽셀
- **범위 데미지**: 폭발 범위 내 모든 대상에게 피해
- **자폭 주의**: 플레이어도 폭발 범위에 있으면 피해를 입음

## 전략 팁

1. **거리 유지**: 너무 가까이서 발사하면 자신의 폭발에 피해를 입습니다
2. **벽 활용**: 벽 뒤에 숨어서 적의 공격을 피하세요
3. **한 방에 여러 적**: 폭발 범위를 이용해 여러 적을 동시에 공격하세요
4. **파워업 확보**: 체력과 공격력 아이템을 적극 활용하세요
5. **움직이면서 발사**: 정지해 있으면 적의 좋은 표적이 됩니다

## 레벨 시스템

- **레벨 1**: 적 3마리
- **레벨 2**: 적 4마리
- **레벨 3**: 적 5마리
- **레벨 4+**: 적 수 계속 증가

레벨이 올라갈수록:
- 적 탱크 수 증가
- 적 탱크 이동 속도 증가
- 적 탱크 발사 빈도 증가

## 기술 스택

- **HTML5 Canvas**: 게임 렌더링
- **JavaScript (ES6+)**: 게임 로직
- **CSS3**: 스타일링 및 UI
- 순수 바닐라 JS (프레임워크 없음)

## 게임 클래스 구조

```javascript
// 주요 클래스
- Game: 게임 전체 관리 (메인 루프, 상태 관리)
- Tank: 탱크 객체 (플레이어/적)
- Bullet: 총알 객체 (이동, 충돌, 폭발)
- Wall: 벽 객체 (파괴 가능/불가능)
- Particle: 파티클 효과 (폭발, 연기)
- PowerUp: 파워업 아이템 (체력, 공격력)
```

## 로컬 실행 방법

1. 모든 파일을 같은 폴더에 저장
2. `index.html` 파일을 브라우저로 열기
3. 게임 시작!

또는 로컬 서버 실행:
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve
```

브라우저에서 `http://localhost:8000` 접속
